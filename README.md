# Smart Ingredient Label

Analyzes food labels for ingredients and groups them into sustainable/health-aware food groups.

## Prerequisites
### Android
* Make sure you have [Android Studio](https://developer.android.com/studio/index.html) and Android SDK installed
* If you want to use your phone instead of the emulator, download and install [Expo](https://expo.io/) 

## Instructions
* After your install the prerequisites, use `npm install` and then `npm start` to start the application
* If you installed Expo, use the QR reader to scan the code in your terminal
